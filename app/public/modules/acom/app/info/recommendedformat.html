Recommended File Formats

Any file format can be submitted to the Academic Commons. However, the file formats below are strongly recommended.

These formats are widely-used, well-understood and stable. They facilitate long-term preservation efforts, which include migration to future file formats when possible.

Textual Formats Extensions
PDF/A .pdf
Other PDF formats .pdf
Plain Text (US-ASCII, UTF-8) .txt
XML
Includes EPUB, XHTML, OpenDocument Format		 .xml, .epub, .od
Comma-Separated Values .csv
Image / Graphic Formats
TIFF .tif, .tiff
JPEG 2000 .jp2
JPEG .jpg, .jpeg
Audio Formats
AIFF .aiff, .aif, .aifc
WAVE .wav
Video Formats
MPEG-2 / MPEG-4 AVC .mpeg, .mpg, .mpe, .mp4

For a more complete list of supported formats, see the Academic Commons Preservation Policy.

If you have questions regarding these file recommendations please contact Sara Amato, Digital Asset Management Librarian.
